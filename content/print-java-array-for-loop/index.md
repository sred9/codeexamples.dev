This below example shows you how to loop through an array in java by using for loop.

```java
String[] arr = {"passion", "fun", "sun", "lab"};
for (int i = 0; i < arr.length; i++) {
  System.out.println(arr[i]);
}
```

Output:

```java
passion
fun
sun
lab
```
