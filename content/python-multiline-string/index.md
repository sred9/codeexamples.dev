We can use triple quotes `""" """` to create a multi-line string in python.

Here is an example

```python
a = """you can type
very long string inside this
quotes
enjoy !!!
..."""

print(a)
```

Output:

```python
you can type
very long string inside this
quotes
enjoy !!!
```
