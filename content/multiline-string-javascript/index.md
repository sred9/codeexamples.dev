We can use backticks `(``)` to create mutline strings in JavaScript.

Here is an example:

```js
const mutlilineString = `you see mee in
Multiple lines
a
b
c
d
e
f
g
`
```

>Note: The backticks `(``)` syntax is also called template literals or template strings which is added in es6.
