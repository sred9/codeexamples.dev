This example will show you how to write a hello world program in Rust

```rust
fn main() {
    //it Prints hello world
    println!("Hello World!");
}
```

## Compiling code

To compile and run the above code use the following command.

```bash
rustc hello.rs
```

It will print the `Hello World!` to the console.
