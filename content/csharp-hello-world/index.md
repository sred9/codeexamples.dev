In this example, you learn about how to print a "Hello world" program in C#.

```csharp
using System;

namespace HelloWorld
{
    class Hello {
        static void Main(string[] args)
        {      // prints hello world
            System.Console.WriteLine("Hello World!");
        }
    }
}
```
